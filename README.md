# 🛍️ Madhav Store Sales Analytics Dashboard

## 📌 Overview
The **Madhav Store Sales Analytics Dashboard** is an interactive business intelligence solution designed to analyze sales performance, customer behavior, and product trends. This project transforms raw retail data into actionable insights, enabling better decision-making and performance tracking.

---

## 🧠 Project Summary
Designed and developed a dynamic sales dashboard to evaluate key business metrics such as revenue, profit, and product performance. Applied data cleaning, transformation, and visualization techniques to derive meaningful insights. This project demonstrates strong analytical skills, business understanding, and data storytelling capabilities.

---

## 🛠️ Tech Stack
- **Visualization Tool:** Power BI   
- **Data Handling:** CSV   

---

## 📂 Data Source
- Retail sales dataset (Madhav Store)
- Includes attributes such as:
  - Order ID  
  - Product Category  
  - Customer Name  
  - City  
  - Sales Amount  
  - Profit  
  - Order Date  

---

## 🚀 Features
- 📊 Interactive dashboard with filters and slicers  
- 💰 Revenue and profit analysis  
- 🛒 Product category performance tracking  
- 🌍 Region-wise sales distribution  
- 🔍 Clean, intuitive, and user-friendly interface
